- Descriptive:
    - Describe a set of data
    - Tries to describe or summarize some data

- Exploratory:
    - Look at data, find relationships
    - Correlation between variables etc.
    - Correlation =!> causation
 
- Inferential:
    - Most common goal of statistical models
    - Estimate quantity of variable of interest
    - Estimate error in quantity

- Predictive:
    - If X predicts Y, it does not mean X causes Y
    - Use training data to predict on new data

- Causual:
    - What causes a change in a particular variable
    - Usually done in terms of averages

- Mechanistic:
    - Very hard to infer
    - Understand exact changes in variables that lead to changes in other variables
